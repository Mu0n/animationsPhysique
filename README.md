# animationsPhysique
Animations (svg) en physique créées et utilisées par le département de physique du cégep de Ste-Foy


## Pour voir les fichiers swf (flash) avec Opera (devrait fonctionner avec Chrome aussi)

Il suffit de faire une page html qui ne contient que cette balise:

<embed type="application/x-shockwave-flash" src="" width="100%" height="100%">

où src devra contenir l'adresse du fichier swf.
